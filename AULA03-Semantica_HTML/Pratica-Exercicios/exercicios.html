Entendendo HTML e Semânica

O HTML (HyperText Markup Language) é a linguagem de marcação utilizada para estruturar o conteúdo das páginas da web. Com ele, é possível organizar textos, imagens, links, listas, tabelas e outros elementos que aparecem em um site. O HTML funciona por meio de tags, que indicam ao navegador como cada parte do conteúdo deve ser interpretada e exibida. Por exemplo, existem tags para títulos, parágrafos, imagens e links, permitindo que a informação seja apresentada de forma organizada e compreensível para o usuário.

A semântica no HTML refere-se ao uso de tags que possuem significado claro sobre o tipo de conteúdo que representam. Em vez de utilizar apenas elementos genéricos, como div, o HTML semântico utiliza tags como header, nav, main, section, article e footer. Essas tags ajudam a descrever melhor a estrutura da página, facilitando a leitura do código por desenvolvedores, melhorando a acessibilidade para tecnologias assistivas e contribuindo para que mecanismos de busca entendam melhor o conteúdo do site. Dessa forma, a semântica torna o código mais organizado, acessível e eficiente.

Tipos de tags semânticas:

    • theader: Utilizamos para representar o cabeçalho de um documento ou seção declarado no HTML. Nele podemos inserir elementos de (h1) a (h6);
      main: Especificamos o conteúdo principal e, consequentemente, de maior relevância dentro da página;
      nav: Utilizamos quando precisamos representar um agrupamento de links de navegação, que, por sua vez, são criados com os elementos (ul), (li) e (a)
      section: Representamos uma seção em um documento e geralmente contém um título, o qual é definido por meio de um dos elementos entre (h1) e (h6).
      aside: Utilizado quando precisamos criar um conteúdo de apoio/adicional ao conteúdo principal;
      figure: É uma marcação de uso específico para a inserção de uma figura;
      footer: Representamos o rodapé de um documento, como a área presente no final de uma página web.
