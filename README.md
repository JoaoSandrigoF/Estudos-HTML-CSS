# 🎯 Objetivo deste repositório

- Cria uma bese para estudos em HTML
- Estudo da estrutura - do mais básico ao mais avançado
- Aplicação prática de dos conhecimentos teóricos
---
## 📖 Conteúdos - Organização
- Os conteúdos ficarão organizados na seguinte forma
    1. 📂 AULA00- NomeDaAula 
        - 1.1- 📂(quando houver) - Pasta com imagens utilizadas em toda pasta ou subpasta da aula 
        - 1.2- 📂(quando houver) - Pasta com páginas HTML - páginas que estão relacionadas ao index.html principal da aula      
        - 1.3- 📂(quando houver) - Pasta com atividades Praticas  - Aplicação prática dos conceitos estudados
            - 1.3.1 - idex.html
            - 1.3.2 - styles.css
            - obs.: as imagens que serão utilizadas nas atividades práticas ficarão na pasta imagens dentro da apasta principal da aula. O objetivo e gerenciar todas a imagens de cada aulas em uma única pasta 
    ---      
    2. Arquivo index.html
        - Com a base conceitual do que foi estudado
    ---
    3. Arquivo styles.css (quando houver)
        - Estilos utilizados na página index.html
    ---
    4. Tema_README.md
        - Aqui ficarão todas as anotações de cada aula;
        - Resumo;
        - Conceitos;
        - O que mais for relavante para compreenção do conteúdo
